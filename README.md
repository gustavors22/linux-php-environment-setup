# linux-php-environment-setup
Script for install php web dev enviroment

## Tools

- PHP 7.4 
- Composer
- NVM
- Node LTS
- Yarn
- Docker
- Visual Studio Code
- DBeaver


##  How to Execute

Download the file from the repository and run the following commands in the folder where the file is located.

```bash
  sudo chmod +x setup.sh
  sudo ./setup.sh 
```
    
## Compatible Systems
- Ubuntu based
